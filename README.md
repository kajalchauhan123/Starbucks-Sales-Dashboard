# Starbucks-Sales-Dashboard
This project presents an interactive Starbucks Sales Dashboard built in Power BI, based on a detailed transactional dataset of 12,000+ rows.
The dashboard provides actionable insights into sales performance, customer behavior, and store-level trends across multiple regions and cities.
